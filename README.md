## Creating Repo

```
    mkdir boilerplate-api
    cd boilerplate-api
    got mod init example/boilerplate-api
```

## After importing a new package

```
    'go mod tidy' or 'go get .'
```

## Starting Repo

```
'go run .'
```

## Generate SQL

```
'sqlc generate'
```

## Committing to Github (and deploying API)

```
'git add .'
'git commit -m ""'
'git push origin main'
```
